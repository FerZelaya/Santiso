# Santiso — Landing Page

Landing page for **Santiso**, conservas artesanales *Deliciosamente Natural*.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4 (design tokens in `@theme`, utilities in components)
- `clsx` + `tailwind-merge` via `cn()` helper

## Development

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Content

All copy, products, and prices come from `idea.md`. Product images are in `public/images/`.
